Mini Casino Client

A public browser entry point for Blackjack Lab. Check the live connection, choose a player name, and enter the multiplayer slots floor without downloading the private server package.

Multiplayer Slots Walk the generated casino floor, see other players, emote, inspect cabinets, and sit at games.
Blackjack Table Practice, learn strategy, and jump from table areas on the floor into blackjack play.
Public Safe This repo ships only browser code. Server tokens and host config stay private.
Live Connection Reads a connection manifest published by the host server at startup.

Connect And Play

  1. Wait for the connection status to show the current server.
  2. Enter the name other players should see.
  3. Click Enter Casino to open directly into Slots multiplayer.

How The Connection Works

The private host server publishes a tiny public connection.json. This page reads that file, finds the latest WebSocket server URL, and passes it to the game client.

1. HostRuns the server and updates the manifest.
2. ManifestStores only status, updated time, and wsUrl.
3. PlayerLaunches the game already connected.

Where Multiplayer Matters

Multiplayer presence is centered on the slots floor: players see each other walking around, emote, gather near cabinets, and move between slot machines and blackjack table areas. Shared blackjack rooms are available from the Multiplayer tab.
